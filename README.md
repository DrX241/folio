# Portfolio EMI — Tech Lead (Next.js)
- Next.js App Router, CSS (no Tailwind)
- Pages: Accueil, Projets (+Détail), Médias, À propos
- Contenu éditable dans `lib/projects.js` (phase 1) puis `/content/projects/*.md` (phase 2)

## Dev
npm install
npm run dev

## Build
npm run build && npm run start

## Deploy
- Vercel recommandé
- Configurer NEXT_PUBLIC_SITE_URL et domaine
